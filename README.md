# 泸州油纸伞品牌展示页

> 一个展示泸州油纸伞文化与产品的品牌形象页面，融合非遗文化元素与现代网页交互设计。

## 项目简介

本项目是一个静态品牌展示网页，以“泸州油纸伞”为主题，展示其作为国家级非物质文化遗产的工艺特色、经典款式和品牌理念。页面设计融合了中国传统美学与现代响应式布局，适合用于文化品牌推广、电商产品展示等场景。

## 技术栈

- **HTML5**：语义化标签，结构清晰
- **CSS3**：Flex/Grid 布局、过渡动画、伪类效果
- **JavaScript**：交互功能（下拉菜单、图片悬停替换等）
- **视频/音频嵌入**：背景视频循环播放、背景音乐

## 项目结构

├── index.html # 主页面
├── css/
│ └── index.css # 样式文件
├── images/ # 图片资源
├── video/ # 背景视频
└── audio/ # 背景音乐

## 页面功能模块

| 模块 | 功能说明 |
|------|--------|
| **顶部导航栏** | Logo + 登录/注册入口 + 三级分类菜单（鼠标悬停展开） |
| **搜索栏** | 商品/品牌/编号搜索入口 |
| **视频背景区** | 自动循环播放非遗文化宣传视频，搭配品牌标语 |
| **经典款式展示** | 白玉兰、桃花、幽兰山谷三款油纸伞介绍 |
| **制伞工艺展示** | 图片悬停切换效果，展示108道工序 |
| **品牌定制区** | 合作品牌展示（欧诗漫、丁家宜等），图片悬停替换 |
| **底部留言表单** | 姓名、邮箱、电话、密码字段 + 留言区 |
| **版权信息** | 底部品牌宣传语 |

## 交互亮点

- **三级菜单下拉**：鼠标悬停“选项”按钮，弹出产品分类、功能分类、节日礼物等导航内容
- **图片悬停替换**：工艺展示区与品牌定制区，鼠标悬停时图片切换，增强视觉反馈
- **表单验证**：电话字段限制11位数字（正则表达式）
- **响应式布局**：适配PC端主流分辨率，部分模块支持移动端
